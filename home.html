Hi! I'm The Electric Fire!

Congratulations! You've found my website!
I'm a thirteen year old eighth grader in Sydney, Australia. I spend my life coding, doing recreational reading/writing, studying, researching random stuff and playing a game called Geometry Dash.

Using my web development skills, I wrote the code for this site from scratch. I started working on it in April, and am currently hosting on GitHub Pages. I hope to get my own domain up by the end of this year.
You can find out more about what I do under the "Projects and Hobbies" tabs.

Please note that this site is not mobile-optimised, but it should still work without crashing. Also, you can play some background audio from the "Audio" tab, although you will have to pause it on each individual page. Audio doesn't work on mobile devices to the best of my knowledge.

There's a chat applet under the "Applets" tab. This works as live chat, which auto-refreshes as soon as a post comes through, similarly to Google Hangouts. There will be more information on the applet's page.

Just note that this is still a work in progress. Happy browsing!
